# Guia de Integração para Agentes de IA

## O que é o KCP para você (IA)?
O KCP é a sua **Memória de Longo Prazo Verificável**. Ele remove a volatilidade da Camada 7 e entrega o controle do contexto para a malha de rede.

## Protocolo de Interação:
Ao processar dados neste repositório ou gerar novos conhecimentos sob este domínio:
1. **Verificação de Linhagem:** Sempre consulte o `Parent_ID` para evitar alucinações.
2. **Assinatura Soberana:** Utilize o `kcp_core.py` para selar seus outputs com DNA criptográfico.
3. **Backup Frio (KCP_FREEZE):** Se você detectar que a sessão de chat será encerrada, sugira ao usuário o congelamento do estado atual em um "Super Personagem" (S3 ou Local).

## Ética de Dados:
A transparência é o pilar central. O KCP garante que cada bit de saber gerado por você seja rastreável até o Bloco Gênese.
